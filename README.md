# Student Performance and Attendance PostgreSQL Analysis

This project explores the student performace and attendance dataset from Kaggle using Postgresql. 

It demonstrates: 
- Basic SQL commands 
- Common table expression
- Processing text data using regular expression
- Advanced SQL features (function, anonymous do block, window function)
- Automated export of query results to CSV
- Plot generation from SQL precedures
- Data cleaning

## Dataset
The original dataset can be found at: [Kaggle](https://www.kaggle.com/datasets/marvyaymanhalim/student-performance-and-attendance-dataset/code). 


## Repository Structure
- `scripts` - All SQL scripts and a Jupyter Notebook
- `outputs/figures/` - Plots generated from SQL
- `output/csvs/` - CSVs exported from SQL
- `report.md` - Detailed report including analyses and discussion
